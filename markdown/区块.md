#### <center0>Markdown区块</center>
## 1、区块使用
> ##### markdown区块引用是在段落开头使用">"符号，后面紧跟一个空格
```
> 区块引用
```
##### 效果：
> 区块引用

## 2、区块嵌套
> ##### 区块是可以嵌套的，一个">"是最外层，两个">"是第一层的第一个嵌套，以此类推
```
> 最外层
> > 第一层嵌套
> > > 第二层嵌套
```
##### 效果：
> 最外层
> > 第一层嵌套
> > > 第二层嵌套

## 3、区块中使用列表
```
> 区块中使用列表
> 1. 第一项
> 2. 第二项
> + 第一项
> + 第二项
```
##### 效果：
> 区块中使用列表
> 1. 第一项
> 2. 第二项
> + 第一项
> + 第二项

## 4、列表中使用区块
```
* 第一项
    > 第一个区块
    > 第二个区块
* 第二项
```
##### 效果：
* 第一项
  > 第一个区块
  > 第二个区块
* 第二项
