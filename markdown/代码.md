#### <center>Markdown代码</center>
## 1、段落中的使用
> ##### 如果是段落上的一个函数或者片段的代码可以用"`"号包裹起来
```
`printf()`函数
```
##### 效果：
`printf()`函数

## 2、代码区块
> ##### 代码区块使用`四个空格`或者一个`制表符（Tab键）`
```
    <?php>
```
##### 效果：
    <?php>

> ##### 或者使用(```)包裹一段代码，并指定一种语言（也可以不指定）
```
假装三个"-"是"`"
---javascript
    alert("fun")
---
```
##### 效果:
```
alert("fun")
```
