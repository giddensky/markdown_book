## 1、锚点定位
> ##### 创建命名锚记,创建到该命名锚记的链接。
```
[说明文字](#jump)
<span id = "jump">跳转到的位置</span>
```
[点击一下](#jump)  
锚点是是网页制作中超级链接的一种，又叫命名锚记。
使用命名锚记可以在文档中设置标记，这些标记通常放在文档的特定主题处或顶部。然后可以创建到这些命名锚记的链接，这些链接可快速将访问者带到指定位置。
创建到命名锚记的链接的过程分为两步。 
1、创建命名锚记 
2、创建 到该命名锚记的链接。

反正就是用于文章快速定位，点击目录便可直接跳转到指定位置的。
